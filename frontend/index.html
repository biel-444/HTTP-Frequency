Como Funciona

01

Entrada de URLs

Insira uma ou mais URLs no campo de texto, uma por linha. O sistema aceita endereços com ou sem https:// e remove duplicatas automaticamente.

02

Execução Assíncrona

O backend em FastAPI dispara todas as requisições de forma concorrente usando asyncio e httpx, respeitando o limite de concorrência configurado via semáforo.

03

Coleta de Métricas

Para cada URL são medidos: tempo de resposta com perf_counter, código HTTP, bytes recebidos, URL final após redirecionamentos e erros de rede.

04

Ranking e Relatório

Os resultados são ordenados por tempo de resposta, gerando um ranking das URLs mais rápidas. Os dados são persistidos no Supabase e exportados em CSV para download.

Sobre o Projeto

O HTTPFrequency foi criado para facilitar testes HTTP, monitoramento de APIs e visualização de respostas em uma interface moderna e intuitiva.

O objetivo do projeto é fornecer uma plataforma moderna para execução concorrente de requisições HTTP, análise de desempenho com a biblioteca HTTPX e geração de métricas e relatórios.
