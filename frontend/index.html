Sobre o Projeto

O HTTPFrequency foi criado para facilitar testes HTTP, monitoramento de APIs e visualização de respostas em uma interface moderna e intuitiva.

O objetivo do projeto é fornecer uma plataforma moderna para execução concorrente de requisições HTTP, análise de desempenho com a biblioteca HTTPX e geração de métricas e relatórios.
